    Overlay fonts

    machine-wide font library for the Overlay Builder — shared across all leagues
    Add font

    A curated set of free Google Fonts ships with every install and is offered in the Overlay Builder's font pickers. Add any other family by name above — it is downloaded once into runtime/fonts/. When a league's design uses a font, it is copied into that league's overlay/fonts/ on save, so the overlay works offline and profile export stays self-contained.

    KeyValue

    Machine-level .env on this computer (ports, OBS-WebSocket password, Companion path) — shared across all leagues. League config (Sheet, OBS collection, …) lives under Profile. Values are masked; comments are preserved. Changes apply when you next (re)start the affected service.

    Crew Console

    role-adaptive crew links — commentator, director & producer

    Zero-config: the relay serves the role-adaptive /console launcher automatically — a per-league secret is generated on first relay start. Every request is token-gated (links below). It is exposed publicly only when you also turn the Funnel on.

    Public access (Tailscale Funnel)

    —
    MagicDNS not available (is Tailscale up?)

    Exposes only /console publicly. Needs MagicDNS + HTTPS + the funnel nodeAttr in the tailnet policy (one-time admin step). Auto-enable is on by default (opt-out) — the Funnel is the preferred produce path; uncheck to keep the relay tailnet/loopback-only.

    Crew links

    Funnel link is the public HTTPS link (needs the Funnel ON above). Internal link reaches the relay over Tailscale (the producer's tailnet IP, or 127.0.0.1 when the tailnet is down) without the Funnel — same rule as the producer panel link.
