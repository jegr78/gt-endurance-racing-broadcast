GT Endurance Racing — Cheat Sheets

Print & pin for event day

Streamer

You stream your stint. That's it.

Settings (set once)

  • Stream to your own fixed channel, same one every event, set to Unlisted (the league tells you which channel).
  • Latency: Low — not Ultra-low (unless your line is rock-solid).
  • Resolution: 1080p target, never below 720p.
  • Encoder: hardware (NVENC / QuickSync / AMF). CBR, keyframe 2 s.
  • Audio: 128–160 kbps AAC, 48 kHz, stereo.
  • No personal overlays/graphics — added centrally.
  • PlayStation direct works (PS5 1080p / base PS4 720p, Unlisted on-console). The console can't set latency — set Low beforehand in YouTube Studio / Twitch dashboard.

On the day

  1. Go live ~5 min before your stint.
  2. Tell the Director you're live.
  3. Stay live through the ~10 min handover, then you can stop.

Producer

Set up, then only start & stop the broadcast.

Before the event

Easiest in the Control Center (double-click racecast-ui); CLI shown below.

  1. Update: racecast update, then racecast install-tools --update.
  2. Update GPU driver; reboot the PC.
  3. Refresh cookies: racecast cookies firefox (log into YouTube in Firefox first). If any stint uses a gated Twitch feed, also run racecast cookies twitch firefox.
  4. Check the machine: racecast preflight — fix anything it flags.
  5. Bring everything up: racecast event start — Tailscale, Discord, relay, OBS, Companion; prints the director URLs to forward.
  6. Companion green; a Director confirms access. Test the Discord audio source.
  7. Enter the league's stream key in OBS.

Run

▶ Start Streaming

  • Hand the show to the Director — you don't touch scenes.
  • Guest feed needed? yt-dlp -g "URL" → give the link to the Director.

Interviews (Discord) — last part only

  • Interviews run at the end of the broadcast over Discord voice.
  • Only the producer of the LAST part does this. 8 h event = 1 part = always the last part → that producer always joins. 12 h / 24 h: only the final-part producer; earlier producers skip Discord entirely.
  • The capture taps your local Discord app, so you must join the "Interviews" voice channel yourself — the Director can't do this remotely.
  • Join before race end (you stay muted in OBS until the Director fires → INTERVIEW). Keep Discord windowed, not fullscreen.

End

  • ■ Stop Streaming.
  • Shut down services: racecast event stop.

Director

Run the show from your browser.

Connect

  1. Panel: http://<producer-tailscale-ip>:8088/panel,
    or Web Buttons: http://<producer-tailscale-ip>:8000/tablet.
  2. Panel scene/audio control only: enter the producer's Tailscale IP + port 4455 + the OBS WebSocket password → Connect (asked once — the browser remembers it). Web Buttons need no password.

During a stint

  • Scene Stint, correct feed up, HUD on.
  • Toggle graphics & volumes as needed.
  • Weather: Weather Race (1) Toggle / Weather Race (2) Toggle / Weather Quali Toggle — full-screen overlays, each an independent toggle.

Handover (every 2 h)

  1. Incoming streamer goes live → their feed appears automatically.
  2. Switch to Splitscreen for ~10 min.
  3. Switch to Stint on the new feed.

Driver POV (optional)

  • Needs lead time (~5 min): driver goes live + sends URL well before you want it on air.
  • URL in Sheet tab POV (row 2; the name column = on-screen label, ≤20, also settable in the panel POV row) → POV Reload → wait until the panel health line (or /status) shows serving → POV Toggle (relay action; PiP bottom-right).
  • Muted by default; MUTE POV / VOL POV UP/DOWN for sound. Done → POV Toggle (hide) + POV Stop.

Intro / Outro

  • INTRO / OUTRO (Page 1, top row) — plays the looping intro/outro clip with audio; press another scene button to leave.

Interview

  • Fire → INTERVIEW (scene Interview + Discord audio up, feeds down).
  • Producer must be joined to the Discord "Interviews" voice channel for audio — confirm with them before you cut.

If a feed stalls

  • Cut to another feed/Standby, tell the Producer, check that streamer is still live.

Control Center racecast-ui

Run the whole station from your browser. Launch: double-click racecast-ui (Linux ./racecast-ui) → http://127.0.0.1:8089. Every action is also a racecast … command.
